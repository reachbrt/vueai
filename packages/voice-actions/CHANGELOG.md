# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-02

### Added
- Initial release of @aivue/voice-actions
- VoiceActions component with speech recognition
- AI-powered natural language command processing
- Custom command registration and execution
- Voice feedback with text-to-speech
- Multi-language support
- Wake word detection
- Continuous listening mode
- Command history and analytics
- Customizable UI with themes
- Keyboard shortcuts
- Accessibility features
