# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-02

### Added
- Initial release of @aivue/doc-intelligence
- Document upload component with drag-and-drop support
- OCR text extraction using Tesseract.js (offline)
- PDF text extraction support
- AI-powered document type detection (invoice, receipt, form, ID)
- Entity recognition (dates, amounts, names, addresses, emails, phone numbers)
- Auto-generated Vue forms from extracted data
- Document viewer with extracted data overlay
- Multi-document processing support
- TypeScript support with full type definitions
- Comprehensive documentation and examples
