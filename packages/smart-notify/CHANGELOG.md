### Features
- **UrgencyDetector**: NLP-based urgency classification
  - Keyword analysis (critical, urgent, ASAP, etc.)
  - Sentiment analysis
  - Time relevance detection
  - Contextual importance scoring
  - Training capability for improved accuracy

- **TimingPredictor**: ML-based optimal timing
  - Historical interaction tracking
  - Best hour identification
  - User behavior learning
  - Focus-aware scheduling
  - DND mode respect

- **GroupingEngine**: Semantic notification grouping
  - TF-IDF text similarity
  - Category and source matching
  - Tag-based grouping
  - Time proximity grouping
  - Automatic group title generation

- **AttentionMonitor**: User attention detection
  - Mouse and keyboard activity tracking
  - Page visibility monitoring
  - Typing state detection
  - Idle time tracking
  - Focus duration measurement

- **BatchingSystem**: Smart notification batching
  - Configurable batch intervals
  - Max batch size limits
  - Priority-based batching rules
  - Automatic batch delivery
  - Never batches critical notifications

- **NotificationCenter Component**: Full-featured UI
  - Floating toggle button with badge
  - Expandable notification panel
  - Priority-based styling
  - Group expand/collapse
  - Attention state display
  - Statistics footer
  - Action buttons
  - Smooth animations

- **useSmartNotify Composable**: Main API
  - Reactive state management
  - Comprehensive notification methods
  - Statistics and analytics
  - Data persistence
  - Export/import functionality
